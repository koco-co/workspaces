    Create `.claude/skills/using-qa-flow/scripts/init-wizard.mjs` with:

    **Imports:** `node:fs` (readdirSync, statSync, existsSync, readFileSync), `node:path` (join, basename, resolve, dirname), `node:url` (fileURLToPath). Dynamic import for `jszip` (only when parsing .xmind).

    **Helper: `inferModuleKeyFromFilename(filename)`** — strips date prefix (`/^\d{6,8}[-_]/`), version suffix (`/[-_]?v?\d+[\d.]*$/i`), file extension (`.csv`, `.xmind`). Returns trimmed string.

    **Helper: `findFilesRecursive(dir, extensions)`** — recursively walks directory, returns `[{ path: relativePath, type: extension }]` for files matching any extension in the list. Used for cases/history/ scanning.

    **Function: `scanProject(rootDir)`** — implements 5-signal scanning per D-02:
    1. Signal 1: Read `cases/xmind/` subdirectories. For each subdirectory entry, check if it contains version subdirectories matching `/^v\d+/`. Build `modules[]` with `{ key: entry, versioned: hasVersionSubdirs, inferredFrom: 'cases/xmind', paths: { xmind: 'cases/xmind/' + entry + '/' } }`. Also scan `cases/archive/`, `cases/requirements/` for additional module keys not found in xmind.
    2. Signal 2: `hasReposDir = existsSync(join(rootDir, '.repos'))`.
    3. Signal 3: `hasImages = existsSync(imagesDir) && readdirSync(imagesDir).some(f => /\.(png|jpg|jpeg|gif|webp)$/i.test(f))` where imagesDir = `join(rootDir, 'assets', 'images')`.
    4. Signal 4: `historyFiles = findFilesRecursive(join(rootDir, 'cases', 'history'), ['.csv', '.xmind'])`.
    5. Signal 5: Scan `cases/requirements/` recursively for files with version patterns in filenames; extract unique versions via `/\bv(\d+\.\d+\.\d+)\b/i`.
    6. Re-init check: `existingConfig = existsSync(join(rootDir, '.claude', 'config.json')) ? JSON.parse(readFileSync(configPath, 'utf8')) : null`. Wrap in try/catch, set null on failure.

    Return: `{ modules, signals: { hasCasesDir, hasReposDir, hasImages, historyFiles, prdVersionPatterns, existingConfig } }`

    **Function: `parseHistoryFile(filePath)`** — dispatches on extension:
    - `.csv`: Read file, strip BOM (`/^\uFEFF/`), extract first line's first comma-separated field (strip quotes). Also call `inferModuleKeyFromFilename(basename(filePath))`. Return `{ candidates: [headerCandidate, filenameCandidate].filter(Boolean), source: 'csv' }`.
    - `.xmind`: Dynamic `import('jszip')`. Read file as Buffer, `JSZip.loadAsync(buf)`. Try `zip.file('content.json')`, parse JSON, extract `sheets[0]?.rootTopic?.title`. Fallback to `zip.file('content.xml')`, regex extract `<title>([^<]+)</title>` from first topic node. Also call `inferModuleKeyFromFilename(basename(filePath))`. Return `{ candidates: [rootTitle, filenameCandidate].filter(Boolean), source: 'xmind' }`.
    - Other: throw Error with message about unsupported format.

    **CLI Router:** Parse `process.argv` for `--command`, `--root-dir`, `--path`. Switch on command:
    - `scan`: call `scanProject(rootDir || process.cwd())`, `console.log(JSON.stringify(result, null, 2))`.
    - `parse-file`: validate `--path` exists, call `parseHistoryFile(path)`, `console.log(JSON.stringify(result, null, 2))`.
    - Default: print usage help and exit(1).

    **Exports:** `export { scanProject, parseHistoryFile, inferModuleKeyFromFilename }` for test imports.

    Run `cd .claude/skills/using-qa-flow/scripts && npm install` to install jszip.
