**Per D-01:** Create `.claude/shared/scripts/notify.mjs` as a single file containing all functions. Structure (top to bottom):

**1. Imports and loadDotEnv (lines 1-30):**
```javascript
import { request } from "node:https";
import { readFileSync, existsSync } from "node:fs";
import { resolve } from "node:path";
import { fileURLToPath } from "node:url";
import nodemailer from "nodemailer";

```

`loadDotEnv(path)`: Parse `.env` file at `resolve(process.cwd(), ".env")`. For each non-empty non-comment line, split on first `=`, trim key and value, strip surrounding quotes from value. Only set `process.env[key]` if key not already present. Call `loadDotEnv()` at module top level.

**2. httpsPost(url, body) (lines ~35-60):**
Returns a Promise. Uses `new URL(url)` to parse. Sets headers `Content-Type: application/json` and `Content-Length: Buffer.byteLength(data)`. Collects response chunks, tries `JSON.parse`, falls back to raw string. Rejects on `req.on("error")`.

**3. mdToHtml(markdown) (lines ~65-85):**
Simple regex-based conversion (per Research "Don't Hand-Roll" — no marked library):
- `## heading` → `<h2>heading</h2>`
- `### heading` → `<h3>heading</h3>`
- `**bold**` → `<strong>bold</strong>`
- `*italic*` → `<em>italic</em>` (but not inside **)
- `- item` → `<li>item</li>` (wrap consecutive in `<ul>`)
- `\n\n` → `<br><br>`
Wrap result in `<div style="font-family: sans-serif; line-height: 1.6;">...</div>`

**4. getEnabledChannels() (lines ~90-100):**
Per D-07 "has URL = enabled". Return array of channel name strings:
- `"dingtalk"` if `process.env.DINGTALK_WEBHOOK_URL` is truthy
- `"feishu"` if `process.env.FEISHU_WEBHOOK_URL` is truthy
- `"wecom"` if `process.env.WECOM_WEBHOOK_URL` is truthy
- `"email"` if BOTH `process.env.SMTP_HOST` AND `process.env.SMTP_USER` are truthy

**5. buildMessage(event, data) (lines ~105-155):**
Per D-04 standard header + dynamic body.
- Read project name: `loadConfig().project?.name ?? "qa-flow"` (wrap in try/catch, default to "qa-flow" on error)
- `timestamp = new Date().toLocaleString("zh-CN", { hour12: false })`
- Event label map: `{"case-generated":"用例生成完成", "bug-report":"Bug 报告生成", "workflow-failed":"工作流失败", "archive-converted":"归档转化完成"}`
- `title = "[" + projectName + "] " + label`
- `header = "**" + title + "** | " + timestamp`
- Body builder per event type using `data` fields:
  - `case-generated`: `- 用例数：${data.count} 条\n- 输出文件：${data.file}\n- 耗时：${data.duration ?? "N/A"}`
  - `bug-report`: `- 报告文件：${data.reportFile}\n- 摘要：${data.summary ?? "N/A"}`
  - `workflow-failed`: `- 失败步骤：${data.step}\n- 错误原因：${data.reason}`
  - `archive-converted`: `- 文件数：${data.fileCount}\n- 用例数：${data.caseCount}`
  - default: `JSON.stringify(data, null, 2)`
- Return `{ title, markdown: header + "\n\n" + body, html: mdToHtml(header + "\n\n" + body) }`

**6. sendDingTalk(title, markdownText, dryRun) (lines ~160-185):**
Per D-05 keyword handling + DingTalk API:
- Read `keyword = process.env.DINGTALK_KEYWORD`
- If keyword truthy AND `!title.includes(keyword)` → `safeTitle = title + " " + keyword`; else `safeTitle = title`
- Payload: `{ msgtype: "markdown", markdown: { title: safeTitle, text: markdownText } }`
- URL: `process.env.DINGTALK_WEBHOOK_URL`
- If dryRun: `console.log(JSON.stringify({ channel: "dingtalk", url, body: payload }, null, 2))` then return
- Else: `const res = await httpsPost(url, payload)`; check `res.errcode !== 0` → `console.error("[notify] DingTalk error:", res)`

**7. sendFeishu(title, markdownText, dryRun) (lines ~190-215):**
Per D-06 webhook only, `msg_type: "post"` format:
- Payload: `{ msg_type: "post", content: { post: { zh_cn: { title, content: [[{ tag: "text", text: markdownText }]] } } } }`
- URL: `process.env.FEISHU_WEBHOOK_URL`
- If dryRun: print and return. Else: `httpsPost(url, payload)`, check `res.code !== 0` → console.error

**8. sendWeCom(markdownText, dryRun) (lines ~220-240):**
Per D-06 webhook only, native markdown:
- Payload: `{ msgtype: "markdown", markdown: { content: markdownText } }`
- URL: `process.env.WECOM_WEBHOOK_URL`
- If dryRun: print and return. Else: `httpsPost(url, payload)`, check `res.errcode !== 0` → console.error

**9. sendEmail(subject, htmlBody, dryRun) (lines ~245-275):**
Per NOTF-05, nodemailer SMTP:
- Create transport: `nodemailer.createTransport({ host: process.env.SMTP_HOST, port: Number(process.env.SMTP_PORT ?? 587), secure: process.env.SMTP_SECURE === "true", auth: { user: process.env.SMTP_USER, pass: process.env.SMTP_PASS } })`
- Mail options: `{ from: process.env.SMTP_FROM ?? process.env.SMTP_USER, to: process.env.SMTP_TO, subject, html: htmlBody }`
- If dryRun: `console.log(JSON.stringify({ channel: "email", transport: { host, port, secure }, mailOptions }, null, 2))` then return
- Else: `await transport.sendMail(mailOptions)` then `transport.close()` (Pitfall 4 from Research)

**10. dispatch(event, data, { dryRun = false } = {}) (lines ~280-305):**
Per D-01 unified entry, Promise.allSettled (Pitfall 5):
- `const { title, markdown, html } = buildMessage(event, data)`
- `const channels = getEnabledChannels()`
- If channels empty: `console.log("[notify] No channels configured, skipping.")` and return `[]`
- Map each channel to its send function:
  - `"dingtalk"` → `sendDingTalk(title, markdown, dryRun)`
  - `"feishu"` → `sendFeishu(title, markdown, dryRun)`
  - `"wecom"` → `sendWeCom(markdown, dryRun)`
  - `"email"` → `sendEmail(title, html, dryRun)`
- `const results = await Promise.allSettled(promises)`
- For each result: if `status === "rejected"`, `console.error("[notify] Channel failed:", result.reason)`
- Return results
