### 1.1.1. **安全问题台账**

#### 1.1.1.1. 程序描述

本模块用于管理数据安全问题台账。支持安全问题的创建，编辑。台账数据来源于数据运营平台，与外部处理系统联动。
![安全问题台账列表页全貌](../../../../../asset/assets/images/安全问题台账列表页全貌.png)

#### 1.1.1.4. **功能详细设计**

本模块包含以下功能：

安全问题台账列表
![安全问题台账列表页搜索与数据表格](../../../../../asset/assets/images/安全问题台账列表页全貌.png) 

该页面用于集中展示所有安全问题的概览信息，以表格形式呈现问题编号、名称、问题类型（风险评估/安全审计）、状态（未关闭/已关闭）、关联资产、发起人等关键字段。

每条记录右侧配置【详情】【编辑】【删除】操作按钮，便于用户快速查看、修改或移除问题（操作权限依状态动态控制），列表支持分页浏览。

搜索
![安全问题台账搜索筛选区域](../../../../../asset/assets/images/安全问题台账搜索筛选区域.png) 

页面顶部提供结构化搜索区域，搜索字段设计如下：

| 字段       | 输入组件类型  | 备注                    |
| -------- | ------- | --------------------- |
| 问题类型     | 下拉选择框   | 枚举值：“风险评估问题”或“安全审计问题” |
| 行动编号     | 文本输入框   |                       |
| 问题名称     | 文本输入框   |                       |
| 问题编号     | 文本输入框   |                       |
| 问题状态     | 下拉选择框   | 枚举值：【待确认】             |
| 问题发起人    | 文本输入框   |                       |
| 问题发起部门   | 文本输入框   |                       |
| 问题处理人    | 文本输入框   |                       |
| 问题处理单位组织 | 文本输入框   |                       |
| 是否进行处理   | 下拉选择框   | 枚举值：是/否               |
| 问题发起时间   | 日期范围选择器 | 起止日期双控件               |
| 问题更新时间   | 日期范围选择器 | 起止日期双控件               |
| 我发起的     | 复选框     | 默认不选中                 |

字段详见附录8-问题台账。

安全问题新增表单页

该页面用于创建新的安全问题记录。用户可填写问题名称、描述、问题类型（风险评估/安全审计）、关联状态报告、是否进行处理及其方案或原因，指定处理人及所属单位组织。
 ![安全问题新增表单页](../../../../../asset/assets/images/安全问题新增表单页.png)

【上传文件】按钮：允许用户上传与质量问题相关的附件，补充问题背景信息。

【新增】按钮：点击后可添加与当前状态报告的关联任务。

所有信息点击【保存】按钮时生效并保存。

状态报告进程字段：

| 字段名称   | 数据类型     | 说明          |
| ------ | -------- | ----------- |
| 行动编号   | 文本       | 行动编号，由表单输入  |
| 行动描述   | 文本       | 行动描述        |
| 计划完成时间 | datetime | 计划完成该行动的日期  |
| 责任部门   | 文本       | 负责执行该行动的组织  |
| 责任人    | 文本       | 行动负责人姓名     |
| 生成时间   | datetime | 上传或创建该方案的时间 |
| 关闭时间   | datetime | 行动实际关闭的时间戳  |

字段详见附录8-问题台账。

安全问题编辑表单页
![安全问题编辑表单页](../../../../../asset/assets/images/安全问题编辑表单页.png) 

提供【保存】【上传文件】和【新增】按钮。

【结束任务】：当问题未未关闭状态时可手动关闭任务。

未通过校验
![安全问题编辑表单校验失败提示](../../../../../asset/assets/images/安全问题编辑表单校验失败提示.png) 

提示框弹窗：在“问题描述”下方出现提示“您好，请重新检查填写字段”，表示当前存在校验失败或必填项缺失，1000ms后自动隐藏。

安全问题详情页
![安全问题详情页只读展示](../../../../../asset/assets/images/安全问题详情页只读展示.png)

展示问题详细信息（只读）：基础信息、附件、状态报告进程、日志等。

日志：记录了对台账的每次操作，包括操作类型、执行人员和操作时间，用于追溯数据变更历史，在创建时插入操作=发布的日志记录。后续任意标记操作保存，插入操作=编辑的日志记录。
