## 商品管理

### 商品列表页

##### 【P0】验证价格保留两位小数显示

> 前置条件
```
1、系统中存在价格为 199.90 的商品"2026春季新款运动鞋"。
2、部署 hotfix/v1.0.1 分支代码。
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 进入【商品管理 → 商品列表】页面 | 页面正常加载 |
| 2 | 找到商品"2026春季新款运动鞋"，查看价格列 | 价格显示为"199.90"（保留两位小数），不再显示为"199.9" |

| 编号 | 步骤 | 预期 |
