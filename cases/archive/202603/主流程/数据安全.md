> 前置条件
```
已有数据源「测试数据源_Doris」且连接正常
已执行以下SQL初始化测试数据：
CREATE DATABASE IF NOT EXISTS test_db;
USE test_db;
CREATE TABLE IF NOT EXISTS user_info(id BIGINT, name VARCHAR(100)) ENGINE=OLAP DUPLICATE KEY(id) DISTRIBUTED BY HASH(id) BUCKETS 1;
已有测试用户「testUser01」
```

| 编号 | 步骤 | 预期 |
| --- | --- | --- |
