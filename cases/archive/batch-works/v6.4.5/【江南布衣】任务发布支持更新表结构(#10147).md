## 任务发布/创建发布包 ❯

##### 【P1】验证周期任务/手动任务打包时，通过「添加关联」可正确关联任务中创建的表（sparkthrift）

> 前置条件

```
1)已登录系统，账号具有任务发布权限
2)系统中存在至少2个项目（测试项目test_project_01和生产项目prod_project_01）
3)测试项目test_project_01和生产项目prod_project_01均配置了Hive数据源
4)测试项目test_project_01中存在至少1个周期任务A或手动任务B，且任务中包含CREATE TABLE语句
5) 建表语句如下:
周期任务A
CREATE TABLE IF NOT EXISTS test_source_tableA (
    id INT COMMENT '主键ID',
    name STRING COMMENT '姓名',
    age INT COMMENT '年龄'
)
STORED AS ORC;


手动任务B
CREATE TABLE IF NOT EXISTS test_source_tableB (
    id INT COMMENT '主键ID',
    name STRING COMMENT '姓名',
    age INT COMMENT '年龄'
)
STORED AS ORC;
```
