## 商品管理

### 商品列表页

##### 【P0】验证商品列表页默认加载

> 前置条件
```
1、系统已存在至少 3 条商品数据，包含上架和下架两种状态。
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 进入【商品管理 → 商品列表】页面 | 页面正常加载，默认显示全部商品，列表按创建时间倒序排列，每页显示 20 条 |
| 2 | 观察列表字段展示 | 列表显示：商品名称、商品分类、SKU编号、价格、库存数量、状态、创建时间、操作列 |

##### 【P1】验证按商品名称模糊搜索

> 前置条件
```
1、商品列表中存在商品"2026春季新款运动鞋"和"2026秋季运动鞋"。
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 进入【商品管理 → 商品列表】页面 | 页面正常加载 |
| 2 | 在搜索框输入"运动鞋"，点击【搜索】按钮 | 列表只显示商品名称包含"运动鞋"的商品，共 2 条 |
| 3 | 清空搜索框，点击【搜索】按钮 | 列表恢复显示全部商品 |

##### 【P1】验证按商品分类筛选

> 前置条件
```
1、商品列表中存在分类为"运动鞋"和"休闲鞋"的商品各至少 1 条。
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 进入【商品管理 → 商品列表】页面 | 页面正常加载 |
| 2 | 在「商品分类」下拉框选择"运动鞋" | 列表只显示分类为"运动鞋"的商品 |

### 新增商品页

##### 【P0】验证填写完整表单后成功新增商品

> 前置条件
```
1、当前系统中不存在名称为"2026春季新款运动鞋"的商品。
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 进入【商品管理 → 商品列表】页面，点击【新增商品】按钮 | 打开新增商品页面，表单所有字段为空 |
| 2 | 在新增表单中按顺序填写如下信息：<br>- 商品名称: 2026春季新款运动鞋<br>- 商品分类: 运动鞋<br>- 价格: 299.00<br>- 库存数量: 100<br>- 商品卖点: 轻量缓震跑鞋（非必填，可填写无） | 各字段输入无报错提示 |
| 3 | 点击【保存】按钮 | 页面提示"新增成功"，自动返回商品列表页，列表第一条显示新增商品，商品名称为"2026春季新款运动鞋"，状态为"下架"，SKU编号自动生成格式为 SKU-2026-XXX |

##### 【P1】验证商品名称为空时无法提交

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 进入【商品管理 → 商品列表】页面，点击【新增商品】按钮 | 打开新增商品页面 |
| 2 | 不填写「商品名称」，其余必填字段正常填写（商品分类: 运动鞋，价格: 299.00，库存数量: 100），点击【保存】按钮 | 「商品名称」输入框下方显示红色提示"请输入商品名称"，【保存】按钮保持可点击状态，表单未提交 |

##### 【P1】验证商品名称重复时无法提交

> 前置条件
```
1、系统中已存在商品名称为"2026春季新款运动鞋"的商品。
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 进入【商品管理 → 商品列表】页面，点击【新增商品】按钮 | 打开新增商品页面 |
| 2 | 在「商品名称」输入框输入"2026春季新款运动鞋"，其余必填字段正常填写，点击【保存】按钮 | 页面提示"商品名称已存在，请修改后重试"，表单未提交 |

### 上下架操作

##### 【P0】验证对下架商品执行上架操作

> 前置条件
```
1、商品"2026春季新款运动鞋"当前状态为"下架"，库存数量为 100。
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 进入【商品管理 → 商品列表】页面 | 找到商品"2026春季新款运动鞋"，状态显示为"下架" |
| 2 | 点击该商品操作列的【上架】按钮 | 弹出确认弹窗，提示"确认将商品"2026春季新款运动鞋"上架？" |
| 3 | 点击确认弹窗的【确认】按钮 | 弹窗关闭，列表中该商品状态更新为"上架"，页面提示"操作成功" |

##### 【P2】验证库存为 0 的商品无法上架

> 前置条件
```
1、商品"2026春季新款运动鞋"当前状态为"下架"，库存数量为 0。
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 进入【商品管理 → 商品列表】页面 | 找到商品"2026春季新款运动鞋"，状态为"下架" |
| 2 | 点击该商品操作列的【上架】按钮 | 页面提示"库存为 0 的商品无法上架，请先补充库存"，商品状态保持"下架"不变 |
