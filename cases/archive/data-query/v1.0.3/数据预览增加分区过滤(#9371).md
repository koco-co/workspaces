# 数据预览增加分区过滤(#9371)（XMind）

---

### 验证 RANGE 分区表数据预览默认仅查询最新分区
#### 进入【业务查询】-【SQL查询】-【数据库导航】
- 进入成功
#### 展开 Doris 数据源，选择 catalog → database → 表 doris_test_range
- 表结构信息展示正确
#### 点击「数据预览」按钮
- 仅显示 dt='2024-12-05' 分区数据
#### 手动执行 SQL : SELECT * FROM doris_test_range PARTITION (p202412)
- 返回与「数据预览」一致的数据行
#### 展开 Inceptor 数据源，选择 database → 表 inceptor_test_range
- 表结构信息展示正确
#### 点击「数据预览」按钮
- 仅显示 dt='2024-12-05' 分区数据
#### 手动执行 SQL : SELECT * FROM inceptor_test_range WHERE dt='2024-12-05'
- 返回与「数据预览」一致的数据行
