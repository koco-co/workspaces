# 支持维度和度量的手动调整(#9374)（XMind）
> 来源：zentao-cases/XMind/统一查询/202510-统一查询v1.0.3.xmind

---

### 验证字段右侧图标变更且可展开操作菜单
#### 进入【平台管理】-【数据模型】页面
- 进入成功
#### 编辑模型记录A
- 进入编辑页面
#### 检查模型预览
- 字段名称右侧图标调整 。点击图标可展开操作菜单
#### 点击设置图标
- 支持以下功能：
a.重命名：点击后字段名进入可编辑状态，交互效果与当前重命名逻辑保持一致。
b.类型切换：根据字段当前归属展示不同操作：
—若为维度字段，则显示“转换为度量”；
—若为度量字段，则显示“转换为维度”。

### 验证字段重命名功能正常
#### 进入【平台管理】-【数据模型】页面
- 进入成功
#### 编辑模型记录A-模型预览窗口
- 进入成功
#### 选择维度字段name, 重命名为「姓名」
- 编辑成功
#### 选择度量字段age, 重命名为「年龄」
- 编辑成功
#### 保存配置后, 再次进入编辑页面-模型预览
- 字段名正常回显

### 验证字段自由切换为维度/度量功能正常
#### 进入【平台管理】-【数据模型】页面
- 进入成功
#### 编辑模型记录A-模型预览窗口
- 进入成功
#### 选择维度字段name, 「转换为度量」
- 1) 编辑成功, Toast提示: 转换成功!
2) 字段位置从维度模块变更到度量模块
#### 选择度量字段age, 「转换为维度」
- 1) 编辑成功, Toast提示: 转换成功!
2) 字段位置从度量模块变更到维度模块
#### 保存配置后, 再次进入编辑页面-模型预览
- 字段位置正常回显
