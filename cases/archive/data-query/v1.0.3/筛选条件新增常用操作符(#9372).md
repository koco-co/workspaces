# 筛选条件新增常用操作符(#9372)（XMind）

### 验证规则配置新增规则
#### 进入【业务查询】-【自助取数】页面
- 进入成功
#### 编辑取数记录A
- 进入编辑页面
#### 进入设置过滤器页面
- 进入成功
#### 展开规则下拉框
- 1) 新增六个规则: 大于等于、小于等于、大于、小于、匹配、不匹配
2) 规则位置排序依次为（从上到下）：大于等于、大于、小于等于、小于、等于、不等于、包含、不包含、有值、无值、匹配、不匹配

### 验证「大于等于」过滤规则功能正常
#### 进入【业务查询-自助取数-新建自助取数】页面
- 进入成功
#### 配置如下:
数据模型: A
图表-字段: name, salary
图表-过滤器: salary
过滤规则: 大于等于, 62000
- 配置成功
#### 点击「更新」
- 更新成功, 结果验证: SELECT t1.name, t1.salary FROM left_test t1 INNER JOIN right_test t2 ON t1.id = t2.id WHERE t1.salary >= 62000;
