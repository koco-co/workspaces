## 一、功能变更说明

### 1.1 变更模块

**模块路径**：流程中心 → 流程发起 → 数据源认证流程（及对应流程列表详情页）

### 1.2 变更内容

#### 数据源认证流程 - L3行列表新增回显字段

**【20260309】变更**

在以下两处的 L3 行列表中，额外回显 2 个字段：

| 新增字段      | 数据来源    | 说明                         |
| --------- | ------- | -------------------------- |
| 原本权威数据源   | 6E2I 模块 | 回显当前 L3 节点在 6E2I 中的权威数据源名称 |
| 原本权威数据源编码 | 6E2I 模块 | 回显当前 L3 节点在 6E2I 中的权威数据源编码 |

**涉及页面**：
1. **发起表单**：流程中心 → 流程发起 → 数据源认证流程 → L3 行列表
2. **详情表单**：流程中心 → 流程列表 → 数据源认证流程详情 → L3 行列表

### 1.3 变更前逻辑

L3 行列表中仅回显原有字段，不包含权威数据源相关信息。

### 1.4 变更后逻辑

L3 行列表在原有字段基础上，额外增加以下两列回显（只读，不可编辑）：
- **原本权威数据源**
- **原本权威数据源编码**

## 二、功能详细设计

### 2.1 发起表单 - L3 行列表

#### 数据源认证流程背景说明

用户在本页面只选择 L2。根据选择的 L2 内容，回显其他所有必填项。填写内容详见附件 6-#3-数据源认证流程。

提交时需要做表单的完整性校验（可能回显 null，提交时阻断）、或者重复性校验（当前未办结的同一流程中，不可有重复的 L3）。

流程通过后，将附录 1-#2-L3-数据源认证信息根据表单内容进行维护。

### 2.2 字段规格

| 字段名称 | 字段类型 | 是否可编辑 | 数据来源 | 备注 |

## 三、影响范围

### 3.1 影响的功能点

- 流程中心 → 流程发起 → 数据源认证流程：L3 行列表新增 2 列回显
- 流程中心 → 流程列表 → 数据源认证流程详情：L3 行列表新增 2 列回显

### 3.2 注意事项

- 新增字段仅做回显，用户不可修改
- 两个字段数据来自 6E2I 模块，需确认接口对接
- 发起表单与详情表单均需同步变更
