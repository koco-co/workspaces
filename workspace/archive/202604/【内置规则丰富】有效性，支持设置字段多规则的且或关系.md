#### string类型字段强转double验证

##### 【P1】验证string类型字段配置取值范围&枚举范围规则时系统强转double类型执行校验

> 前置条件
```
1) 使用 admin 账号登录数据资产平台
2) 已创建规则任务 quality_task_15695_str，已关联 Doris 数据表 test_db.quality_test_str
3) Doris 建表及灌数 SQL：
CREATE TABLE test_db.quality_test_str (
  id INT NOT NULL,
  score_str VARCHAR(50),
  category VARCHAR(50)
) DISTRIBUTED BY HASH(id) BUCKETS 3 PROPERTIES("replication_num"="1");
INSERT INTO test_db.quality_test_str VALUES
  (1, '5.5', '2'),
  (2, '15.0', '4'),
  (3, 'abc', '1'),
  (4, '-1.0', '3');
-- score_str 为 VARCHAR 类型，数值字符串可强转为 double，'abc' 无法强转
-- 规则：score_str >1 且 in '5,5.5,15'
-- id=1(score_str='5.5'强转5.5，满足>1且in枚举) 通过
-- id=2(score_str='15.0'强转15.0，满足>1且in枚举) 通过
-- id=3(score_str='abc'强转失败) 不通过
-- id=4(score_str='-1.0'强转-1.0，不满足>1) 不通过
```
