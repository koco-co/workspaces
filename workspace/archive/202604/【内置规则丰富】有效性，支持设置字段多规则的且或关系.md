## 数据质量

### 规则集管理

#### 规则配置-新增取值范围&枚举范围规则（P0核心流程）

##### 【P0】验证在规则集中按完整顺序新建取值范围&枚举范围且关系规则（数值类型字段）

> 前置条件

```
1) 使用 admin 账号登录数据资产平台
2) 已在 Doris 数据源中准备测试表及数据：
DROP TABLE IF EXISTS test_db.quality_test_num;
CREATE TABLE test_db.quality_test_num (
  id INT NOT NULL,
  score DOUBLE,
  category VARCHAR(50)
) DISTRIBUTED BY HASH(id) BUCKETS 3 PROPERTIES("replication_num"="1");
INSERT INTO test_db.quality_test_num VALUES
  (1, 5.0, '2'),
  (2, 15.0, '4'),
  (3, 3.0, '1'),
  (4, -1.0, '3'),
  (5, 8.0, '5');
-- 说明：score 字段期望 >1 且 <10；category 枚举值期望 in '1,2,3'
-- 符合且关系（两者均满足）的记录：id=1(score=5,category=2)、id=3(score=3,category=1)
-- 不符合的记录：id=2(score=15越界)、id=4(score=-1越界)、id=5(category=5不在枚举)
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 进入【数据质量 → 规则集管理】页面，等待规则集列表加载完成 | 规则集管理页面打开，列表显示已有规则集数据行 |
| 2 | 点击【新建规则集】，在 Step 1 基础信息中按页面实际顺序填写：<br>- *选择数据源: Doris<br>- *选择数据库: test_db<br>- *选择数据表: quality_test_num<br>- 规则集描述: 无<br>- *规则包名称: 且关系校验包<br>点击【下一步】 | Step 1 基础信息校验通过，进入 Step 2 监控规则页面 |
| 3 | 在 Step 2 监控规则中，在规则包“且关系校验包”下点击【新增规则】，在统计函数下拉框中选择【取值范围&枚举范围】，并按顺序填写如下：<br>- *字段: score<br>- *取值范围行-第一组: 期望值输入 1，操作符选择 ><br>- *取值范围行-第二组: 点击右侧【+】添加第二条，期望值输入 10，操作符选择 <<br>- *条件关系: 选择【且】单选按钮<br>- *枚举值行-枚举值类型: 选择 in<br>- *枚举值行-枚举值信息: 依次输入 1、2、3<br>- 强弱规则: 强规则<br>- 过滤条件: 无<br>- 规则描述: score取值范围1到10且枚举值in 1,2,3 | 规则配置区域展开正常，字段、取值范围、枚举值、条件关系和强弱规则均可正常录入 |
| 4 | 点击规则行【保存】按钮，再点击页面底部【保存】完成规则集保存 | 规则集保存成功，规则列表中对应规则显示：<br>1) 规则类型显示「取值范围&枚举范围」<br>2) 字段显示 score<br>3) 取值范围显示 >1 且 <10<br>4) 枚举值显示 in '1,2,3'<br>5) 且或关系显示「且」<br>6) 强弱规则显示「强规则」 |
