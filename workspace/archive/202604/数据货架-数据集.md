## 数据货架

### 数据集资源

#### 页面初始化

##### 【P0】验证数据集资源页面初始化加载页签、搜索区、目录树与卡片区

> 前置条件
```
1、已登录系统，拥有【数据货架】访问权限。
2、系统中已存在若干已上架的手工数据集资源。

```

> 用例步骤

| 编号 | 步骤 | 预期 |
| --- | --- | --- |
| 1 | 进入【数据货架 → 数据集资源】页面 | 页面打开成功，顶部页签区显示【作品资源】【数据集资源】【API资源】【指标资源】等页签，【数据集资源】处于选中状态。 |
| 2 | 观察搜索区与统计栏 | 搜索区显示"资源名称"输入框及【搜索】【重置】按钮，右侧显示"编目更新时间"排序入口与"筛选结果：共X个结果"统计。 |
| 3 | 观察左侧数据目录树与右侧卡片区 | 左侧展示 L0-L3 数据目录树，默认展开至 L1 层；右侧加载资源卡片，卡片区无空白、报错或重复渲染。 |

#### 资源卡片

##### 【P0】验证手工数据集资源卡片展示来源标签及核心字段与编目信息一致

> 前置条件
```
1、列表中存在已上架、已发布的手工数据集资源。

```

> 用例步骤

| 编号 | 步骤 | 预期 |
| --- | --- | --- |
| 1 | 进入【数据货架 → 数据集资源】页面 | 页面加载成功，手工数据集卡片可见。 |
| 2 | 查看手工数据集资源卡片 | 卡片头部展示【手工数据集】来源标签；资源名称区显示表的英文名称与中文名称；基础信息区显示关联业务对象（数据目录路径）、资产所属部门、收藏数和浏览数；底部显示编目更新时间及【详情】按钮，各字段值与已发布的编目信息一致。 |
| 3 | 查看带置顶标记的手工数据集卡片（如存在） | 卡片头部额外显示【置顶】标记，且排列顺序优先于非置顶卡片。 |

#### 数据目录

##### 【P0】验证数据目录树默认展开至L1层并支持L2与L3节点展开收起

> 前置条件
```
1、系统已配置完整的 L0-L3 数据目录结构。

```

> 用例步骤

| 编号 | 步骤 | 预期 |
| --- | --- | --- |
| 1 | 进入【数据货架 → 数据集资源】页面 | 数据目录树加载成功，默认展开至 L1 层，L2 及以下节点未全部展开。 |
| 2 | 点击任意 L1 节点旁的展开箭头 | 该 L1 节点展开，可见其下的 L2 子节点。 |
| 3 | 点击 L2 节点旁的展开箭头 | 该 L2 节点展开，显示其下的 L3 子节点。 |
| 4 | 再次点击已展开的 L2 节点旁收起箭头 | L3 子节点收起，箭头状态同步切换为收起方向。 |

##### 【P1】验证选择L3末级目录节点后列表仅展示该节点下的数据集

> 前置条件
```
1、某 L3 目录节点下已有已上架数据集，其他节点下亦有数据集，列表总数量大于该单节点下的数量。

```

> 用例步骤

| 编号 | 步骤 | 预期 |
| --- | --- | --- |
| 1 | 进入【数据货架 → 数据集资源】页面 | 右侧默认展示所有已上架数据集资源。 |
| 2 | 在目录树中展开至某 L3 末级节点并点击选中 | 该节点高亮，右侧卡片立即刷新，结果数量缩减。 |
| 3 | 观察筛选后的资源卡片 | 卡片均属于所选 L3 节点关联的数据集，不显示其他目录节点的资源；筛选结果数量与该 L3 节点下的实际资源数一致。 |

##### 【P1】验证选择非末级目录节点时列表包含本级与所有子级数据集

> 前置条件
```
1、某 L2 目录节点下包含多个 L3 子节点，各 L3 子节点下分别有已上架数据集。

```

> 用例步骤

| 编号 | 步骤 | 预期 |
| --- | --- | --- |
| 1 | 进入【数据货架 → 数据集资源】页面 | 右侧默认展示所有已上架数据集资源。 |
| 2 | 在目录树中点击某 L2 非末级节点 | 该节点高亮，右侧卡片刷新，结果数量缩减。 |
| 3 | 观察筛选结果中的资源范围 | 结果包含该 L2 节点下所有 L3 子节点的数据集；卡片数量等于所有子节点下已上架数据集之和，不多不少。 |

#### 基础搜索

##### 【P1】验证按英文名称与中文名称模糊搜索并更新结果数量

> 前置条件
```
1、已存在若干已上架的手工数据集，其中部分表的英文名包含特定关键词，部分表的中文名包含另一特定关键词。

```

> 用例步骤

| 编号 | 步骤 | 预期 |
| --- | --- | --- |
| 1 | 进入【数据货架 → 数据集资源】页面 | 页面默认显示所有已上架数据集资源。 |
| 2 | 在"资源名称"输入框输入某英文关键词后点击【搜索】 | 右侧卡片按英文名模糊匹配刷新，筛选结果统计更新，仅展示名称包含该关键词的资源卡片。 |
| 3 | 点击【重置】按钮 | 搜索框清空，列表恢复显示全部已上架资源，按默认编目更新时间倒序排列。 |
| 4 | 在"资源名称"输入框输入某中文关键词后点击【搜索】 | 右侧卡片按中文名模糊匹配刷新，仅展示名称包含该中文关键词的资源卡片，不显示不包含该词的资源。 |

##### 【P1】验证目录筛选与名称搜索取AND逻辑共同生效

> 前置条件
```
1、某 L2 目录节点下有 3 条已上架数据集，其中仅 1 条的名称包含特定关键词，其他目录节点下亦有含该关键词的数据集。

```

> 用例步骤

| 编号 | 步骤 | 预期 |
| --- | --- | --- |
| 1 | 进入【数据货架 → 数据集资源】页面 | 页面默认显示所有已上架数据集资源。 |
| 2 | 点击目录树中目标 L2 节点使筛选生效 | 目录筛选生效，右侧仅展示该 L2 节点下的数据集。 |
| 3 | 在"资源名称"输入框输入只属于目录内 1 条资源的关键词后点击【搜索】 | 结果在目录筛选基础上进一步收敛，仅显示同时满足目录条件与名称条件的 1 条资源；不展示目录外名称匹配的资源，也不展示目录内名称不匹配的资源。 |

##### 【P2】验证搜索无结果时显示空状态并统计为0

> 前置条件
```
1、系统中不存在名称包含特定极少见字符串的数据集。

```

> 用例步骤

| 编号 | 步骤 | 预期 |
| --- | --- | --- |
| 1 | 进入【数据货架 → 数据集资源】页面 | 页面默认显示已上架数据集资源。 |
| 2 | 在"资源名称"输入框输入不存在的字符串后点击【搜索】 | 右侧卡片区显示空状态提示，统计区更新为"共0个结果"，不展示任何资源卡片或【详情】按钮，页面无报错。 |

#### 排序

##### 【P1】验证编目更新时间默认倒序排列并支持切换为升序

> 前置条件
```
1、列表中存在多条编目更新时间不同的已上架数据集。

```

> 用例步骤

| 编号 | 步骤 | 预期 |
| --- | --- | --- |
| 1 | 进入【数据货架 → 数据集资源】页面 | 页面默认按编目更新时间从新到旧（倒序）展示资源卡片，首张卡片的编目更新时间最晚。 |
| 2 | 点击"编目更新时间"排序入口切换为升序 | 资源卡片重新排列，改为按编目更新时间从旧到新（升序），首张卡片的编目更新时间最早；当前筛选条件保持不变。 |
| 3 | 再次点击排序入口切换回倒序 | 资源卡片恢复倒序排列，顺序与初始状态一致。 |

### 手工数据集详情页

#### 详情进入

##### 【P0】验证进入手工数据集详情页后查看次数加1

> 前置条件
```
1、已登录系统，拥有【数据货架 → 数据集资源】查看权限。
2、列表中存在已上架且已发布的手工数据集，记录其当前查看次数为 N。

```

> 用例步骤

| 编号 | 步骤 | 预期 |
| --- | --- | --- |
| 1 | 进入【数据货架 → 数据集资源】页面 | 目标手工数据集卡片可见，卡片展示的浏览数为 N。 |
| 2 | 点击该卡片上的【详情】按钮 | 页面成功跳转至手工数据集详情页。 |
| 3 | 观察详情页中的浏览数 | 浏览数显示为 N+1，较进入前增加 1。 |
| 4 | 返回列表页查看同一卡片的浏览数 | 卡片浏览数同步更新为 N+1。 |

#### 编目属性

##### 【P0】验证手工数据集详情页抬头展示编目属性内容与已发布编目信息一致

> 前置条件
```
1、已登录系统，拥有【数据货架】访问权限。
2、已存在已上架且已发布的手工数据集，其编目信息完整。

```

> 用例步骤

| 编号 | 步骤 | 预期 |
| --- | --- | --- |
| 1 | 进入【数据货架 → 数据集资源】页面后点击目标手工数据集的【详情】按钮 | 成功进入手工数据集详情页。 |
| 2 | 观察详情页抬头编目属性区展示内容 | 抬头区展示表名（英文名称）、表中文名称、资源 Owner、所属部门、数据目录路径、资源描述等编目属性，各字段值与已发布的编目信息一致。 |
| 3 | 观察详情页整体页签结构 | 详细信息区仅包含【字段信息】【数据预览】两个 Tab，不展示 DataWorks 专属页签（如技术元数据、分区信息、表变更记录）。 |

##### 【P1】验证详情页有拓展属性时抬头区正常展示拓展属性内容

> 前置条件
```
1、已存在已上架且已发布的手工数据集，该数据集在后台编目页已配置至少一条启用的拓展属性。

```

> 用例步骤

| 编号 | 步骤 | 预期 |
| --- | --- | --- |
| 1 | 进入【数据货架 → 数据集资源】页面后点击目标手工数据集的【详情】按钮 | 成功进入手工数据集详情页。 |
| 2 | 观察抬头区拓展属性展示 | 抬头区拓展属性栏正常展示，各拓展属性名称与属性值与已发布的编目信息一致，不出现空行或乱码。 |

##### 【P2】验证无拓展属性时拓展属性区不展示或显示空状态

> 前置条件
```
1、已存在已上架且已发布的手工数据集，该数据集在后台编目页未配置任何拓展属性。

```

> 用例步骤

| 编号 | 步骤 | 预期 |
| --- | --- | --- |
| 1 | 进入【数据货架 → 数据集资源】页面后点击目标手工数据集的【详情】按钮 | 成功进入手工数据集详情页。 |
| 2 | 观察抬头区拓展属性栏 | 拓展属性区不显示或显示空状态提示，不出现空行、乱码或多余占位符。 |

#### 字段信息

> 前置条件
```
1、已存在已上架且已发布的手工数据集，该数据集已在后台编目页录入字段信息。

```

> 用例步骤

| 编号 | 步骤 | 预期 |
| --- | --- | --- |
| 1 | 进入【数据货架 → 数据集资源】页面后点击目标手工数据集的【详情】按钮 | 成功进入手工数据集详情页。 |
| 2 | 观察详情页下方 Tab 区默认状态 | 详细信息区默认激活【字段信息】Tab，字段列表正常加载，无空白或报错。 |

#### 数据预览

##### 【P1】验证切换至数据预览Tab后展示预览文件卡片并可下载

> 前置条件
```
1、已存在已上架且已发布的手工数据集，该数据集已在后台编目页上传预览文件。

```

> 用例步骤

| 编号 | 步骤 | 预期 |

| 2 | 点击【数据预览】Tab | 页面切换至数据预览区，展示文件卡片，文件卡片数量与后台上传的预览文件数量一致。 |
| 3 | 点击文件卡片上的下载按钮 | 触发文件下载，文件正常保存至本地，不出现报错或权限提示。 |

| 编号 | 步骤 | 预期 |

| 2 | 点击【数据预览】Tab | 数据预览区显示空状态提示（如"暂无预览文件"），不展示文件卡片，页面无报错。 |
