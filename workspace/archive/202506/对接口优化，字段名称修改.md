##### 【P1】验证【字段元模型】-【隐私等级】属性支持修改

> 前置条件

```
无
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 进入【元数据】-【元数据模型】-【字段元模型】页面 | 【编辑】按钮可操作，【删除】按钮置灰 |
| 2 | 查看【隐私等级】属性 | 弹【隐私等级】编辑弹窗 |
| 3 | 点击【隐私等级】-【编辑】按钮 | 弹窗标题: 编辑属性属性名： security_Level - 不可编辑属性中文名： 隐私等级 - 可编辑属性类型： 枚举是否必填: 默认【非必填】- 可编辑应用数据源：全部-不可编辑 |
| 4 | 【隐私等级】弹窗校验 | 编辑成功 |
| 5 | 编辑【属性中文名】-“隐私等级“为“隐私等级_测试” | 编辑成功 |
| 6 | 编辑【非必填】-“非必填“为“必填” | 保存成功， 【属性中文名】/【是否必填】字段值更新成功 |
| 7 | 保存修改 |  |
