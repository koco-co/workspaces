##### 【P1】验证质量概览查看功能

> 前置条件

```
无
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 非管理员/管理员，有概览的查看权限点，进入质量概览，查看 | 可查看所有数据 |
| 2 | 非管理员/管理员，没有概览的查看权限点，进入质量概览，查看 | 不允许查看 |
| 3 | 非管理员点击概览中展示的表名/点击告警次数、告警规则跳转，查看数据展示 | 只能展示有表权限的相关任务 |
| 4 | 管理员点击概览中展示的表名/点击告警次数、告警规则跳转，查看数据展示 | 展示所有数据 |

##### 【P0】验证用户为非管理员角色时，数据质量权限功能

> 前置条件

```
无
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 进入数据质量，查看规则任务 | 只能查看负责的表的相关规则任务 |
| 2 | 查看任务实例 | 只能查看负责的表的相关任务实例 |
| 3 | 查看质量报告 | 单表和综合报告均只能查看负责的表的相关报告 |
