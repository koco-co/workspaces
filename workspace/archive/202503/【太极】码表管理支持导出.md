##### 【P1】验证数据标准->码表管理 验证【导出代码】支持导出筛选后的数据

> 前置条件

```
无
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 1、进入 数据标准->码表管理 页面2、按代码目录/模糊搜索/更新用户进行代码筛选，按 被引用数量/更新时间 排序3、点击【导出代码】按钮4.   点击二次确认 | 导出一个excel文件 |
| 2 | 查看导出文件格式 | 导出文件中拆分为代码信息，编码信息两个sheet页展示 |
| 3 | 查看导出文件-代码信息sheet页面内容 | 1）代码信息sheet页面中正确显示筛选排序后的代码数据。2）代码的代码名称，代码编号，代码目录，代码说明，代码来源，创建时间，更新时间，更新用户数据正常显示 |
| 4 | 查看导出文件-编码信息sheet页面内容 | 1）编码信息sheet页面中显示筛选排序后的编码数据。2）编码信息中的代码名称、代码编号取得代码信息sheet页面中的代码名称、代码编号。3）编码取值、编码名称、编码说明数据显示正确 |

##### 【P0】验证数据标准->码表管理 验证【导出代码】支持导出当前列表下所有数据

> 前置条件

```
1.代码目录A下，有aa,bb两个代码。
代码aa的编码内容有五条，
代码bb的编码内容有1条
2.代码目录C下，有代码cc，代码cc的编码内容有1条
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 1、进入 数据标准->码表管理 页面2、点击【导出代码】按钮3.   点击二次确认 | 导出一个excel文件 |
| 2 | 查看导出文件格式 | 导出文件中拆分为代码信息，编码信息两个sheet页展示 |
| 3 | 查看导出文件-代码信息sheet页面内容 | 1）代码信息sheet页面中有aa，bb，cc三条代码数据。2）aa，bb，cc代码的代码名称，代码编号，代码目录，代码说明,代码来源，创建时间，更新时间，更新用户数据正常显示 |
| 4 | 查看导出文件-编码信息sheet页面内容 | 1）编码信息sheet页面中有7两条编码数据。2）编码信息中的代码名称、代码编号取得代码信息sheet页面中的代码名称、代码编号。3）代码aa对应5个“编码取值、编码名称、编码说明”      代码bb对应5个“编码取值、编码名称、编码说明”      代码cc对应1个“编码取值、编码名称、编码说明”4）编码取值、编码名称、编码说明显示正确 |
