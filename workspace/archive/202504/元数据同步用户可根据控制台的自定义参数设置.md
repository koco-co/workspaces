##### 【P1】验证非自定义元数据同步用户功能

> 前置条件

```
在控制台的SparkThrift的meta计算组件/Hive的meta计算组件未配置自定义参数agent.meta.collect.user为user
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 1.进入【元数据】-【元数据同步】页面2.新建临时同步任务/周期同步任务 | 元数据正常同步 |

##### 【P0】验证自定义元数据同步用户功能

> 前置条件

```
在控制台的SparkThrift的meta计算组件/Hive的meta计算组件配置自定义参数agent.meta.collect.user为user
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 1.给user用户和root用户配置访问hdfs的目录权限2.进入【元数据】-【元数据同步】页面3.新建临时同步任务/周期同步任务 | 元数据正常同步 |
| 2 | 1.没有给user用户配置访问hdfs的目录权限2.进入【元数据】-【元数据同步】页面3.新建临时同步任务/周期同步任务 | 元数据同步失败 |
| 3 | 1.没有给user用户配置访问hdfs的目录权限，给root用户配置访问hdfs的目录权限2.进入【元数据】-【元数据同步】页面3.新建临时同步任务/周期同步任务 | 元数据同步失败 |
