## 规则集

##### 【P1】验证编辑规则集修改「是否发送明细数据」后功能正确

> 前置条件

```
| 已存在单表校验规则A，告警配置如下：
- 告警方式：邮箱
- 是否发送明细数据：否
- 告警接收人：用户A
测试表包含不符合规则的数据
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 进入【数据资产】-【数据质量】-【规则任务配置】页面 | 成功进入【规则任务配置】页面，页面内容正常加载显示，无报错 |
| 2 | 选择规则A，点击【编辑调度属性】按钮 | 弹出「编辑调度属性」弹窗 |
| 3 | 在「是否发送明细数据」字段选择「是」 | 「是否发送明细数据」修改为「是」，选项高亮显示 |
| 4 | 点击【保存】按钮 | 保存成功，系统给出成功反馈 |
| 5 | 选择规则A，点击【立即运行】按钮，等待任务执行完成 | 任务执行完成，校验结果为「不通过」 |
| 6 | 登录用户A邮箱，查看收到的告警邮件 | ZIP附件命名为「不符合规则的明细数据」，解压后包含Excel文件，每个校验不通过的规则生成一个Excel，命名格式为「{规则名称}.xlsx」 |
