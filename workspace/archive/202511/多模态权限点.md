##### 【P1】验证数据资产新增非结构化权限点功能

> 前置条件

```
无
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 进入【数据资产】-【角色管理】，查看角色展示 | 1）新增非结构化权限点，有文件管理、上传记录、文件编目，每个功能有操作和查看权限2）访客没有文件管理、上传记录、文件编目的操作权限，为X，其他的都为✅ |
| 2 | 进入【数据资产】，右上角设置按钮-【角色管理】，查看资产新增权限点功能 | 1）新增非结构化权限点，有文件管理、上传记录、文件编目，每个功能有操作和查看权限2）访客没有文件管理、上传记录、文件编目的操作权限，为X，其他的都为✅ |

##### 【P1】验证数据资产角色按最新菜单调整功能

> 前置条件

```
权限点文档：https://dtstack.yuque.com/rd-center/dataassets/hhd59nsg1489zx1r#3Keq
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 进入【数据资产】-【角色管理】，查看角色展示 | 角色名称、位置、及勾选状况展示和权限点文档一致 |
| 2 | 进入【数据资产】-右上角设置按钮-【角色管理】，查看资产角色展示 | 角色名称、位置、及勾选状况展示和权限点文档一致 |
