##### 【P0】验证测试属性名列的批量编辑功能

> 前置条件

```
已进入元数据-数据地图-表详情页
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 查看该表字段的非分区字段和分区字段详情 | 列表都新增了测试属性名表头， 且列表右下角新增批量编辑按钮 |
| 2 | 查看测试属性名列 | 显示内容，且右边包含支持编辑字段属性的图标 |

| 5 | 正确编辑全部属性名值，点击保存按钮 | 属性名全部成功被修改 |
| 6 | 编辑部分属性名，点击取消按钮 | 编辑取消，属性名未被修改 |
