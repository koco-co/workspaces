> 前置条件

```
1.进入数据安全-分级数据页面，页面存在多条记录
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 点击页面搜索框下拉 | 包含找表选项 |
| 2 | 点击选择找表 | 输入框占位符显示：请输入表名搜索 |
| 3 | 输入部分表名，选择对应数据源类型、数据源、数据库，执行搜索 | 显示对应数据源的数据库下的符合表名的表对应字段数据 |
| 4 | 输入准确表名，选择对应数据源类型、数据源、数据库，执行搜索 | 显示对应数据源的数据库下的该表名的表对应字段数据 |
| 5 | 输入表名，选择与之不相对应的数据源类型、数据源、数据库 | 无数据显示 |
| 6 | 输入特殊字符，执行搜索 | 无数据显示 |
