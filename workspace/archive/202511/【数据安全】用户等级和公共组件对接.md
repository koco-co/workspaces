##### 【P1】验证删除用户等级后对应功能正确

> 前置条件

```
1.公共组件存在级别最高的用户等级名称：L1；

级别第二高的用户等级名称：L2

2.存在用户的级别名称为L1
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 删除L1 | 删除成功，最高级别对应的级别名称为L2 |
| 2 | 点击进入数据安全-级别管理-添加级别页面 | 进入成功 |
| 3 | 鼠标移至开放用户等级右边“？”图标 | 显示提示信息：设置后，归属于该级别的字段只开放给对应用户等级的用户查看，默认开放给最高级别的用户 （L2级别表示最高级别用户，L2级以下等级用户均无查看权限） |
| 4 | 查看[分级管理]列表 | 显示为公共组件自定义的名称：最高级别对应的开放用户等级更改为L2 |
| 5 | 登录级别名称为L1的用户，查看分级数据表的信息 | 无法查看 |
| 6 | 修改该用户级别名称为L2，查看分级数据表的信息 | 可以查看到正确信息 |
