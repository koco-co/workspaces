##### 【P1】验证数据库配置记录删除操作

> 前置条件

```
1.数据安全-自动分级-编辑规则页面已打开
2.已存在一条数据库配置：数据源为X，数据库为Y，数据库过滤为test.*
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 查看已存在的一条记录后方删除按钮 | 置为灰色，不支持删除 |
| 2 | 点击新增按钮 | 下方新出现一行选择框 |
| 3 | 在数据源处选择A，数据库选择B，数据库过滤输入test.* | 输入成功，查看删除按钮置为蓝色，支持删除 |
| 4 | 选中数据源为A的记录，点击删除按钮 | 该条记录成功删除，无需二次确认，删除按钮置为灰色 |
