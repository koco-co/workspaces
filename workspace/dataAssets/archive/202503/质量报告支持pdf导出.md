## 综合报告-->

##### 【P1】验证报告详情导出报告功能

> 前置条件

```
无
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 进入数据资产-数据质量-综合报告-查看报告，查看页面右上角 | 操作列新增导出报告按钮 |
| 2 | 点击导出报告 | 提示“请确认是否导出报告” |
| 3 | 点击确定 | 以pdf形式导出该表的报告，文件命名为“综合质量报告_admin@dtstack.com(表负责人名称).pdf” |
| 4 | 查看导出的pdf文件 | 1）近期规则校验异常明细不展示操作列2）其他和平台内报告详情一致 |
