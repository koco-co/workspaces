##### 【P1】验证建表关联指标/维度，表详情展示

> 前置条件

```
表test_table
1）有字段中文名：用户姓名，英文名：user_name，绑定了指标：用户姓名

2）有字段中文名：姓名1，英文名：name1，绑定了维度：姓名1
```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 进入数据地图，表test_table的详情页，查看字段姓名1、用户姓名 | 字段名后有指标/维度标识的icon |
| 2 | 点击姓名1，查看维度详情弹窗 | 1）详情弹窗展示【基础信息】【维度值】2）具体参数和内容展示与指标库一样 |
| 3 | 点击用户姓名，查看指标详情弹窗 | 1）详情弹窗展示【基础信息】【业务信息】【技术信息】2）具体参数和内容展示与指标库一样 |
