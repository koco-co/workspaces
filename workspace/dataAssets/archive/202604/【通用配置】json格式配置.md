> 前置条件

```
1) 使用 admin 账号登录系统
2) 准备 XLSX 导入文件（5 个 Sheet 齐全，缺少任意层级服务端报"导入文件缺少层级"），文件内容如下：

   | keyA | 中文名A | |

   | illegalParent（故意不存在于一层） | keyB | 中文名B | |
```
