```

```

> 用例步骤

| 编号 | 步骤 | 预期 |
| ---- | ---- | ---- |
| 1 | 进入数据资产-元数据同步，新增元数据同步任务，立即执行，查看状态 | 1）执行过程中，最近一次实例状态为“同步中”2）执行结束后，最近一次实例状态为“同步完成/同步失败” |
| 2 | 将步骤一中的任务，多次点击立即同步，当最后一个实例在同步中时，查看状态 | 同步状态和最近一次实例状态为“同步中” |
| 3 | 当最后一个实例同步完成/同步失败，有其他实例在同步中时，查看状态 | 同步状态为“同步中”；最近一次实例状态为“同步完成/同步失败” |
| 4 | 当最后一个实例同步完成/同步失败，且其他实例都同步结束时，查看状态 | 同步状态为“同步完成”；最近一次实例状态为“同步完成/同步失败” |
